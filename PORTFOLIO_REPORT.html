1. Project Overview

A modern, fully responsive Cybersecurity Portfolio built with React + TypeScript + Vite. The portfolio showcases Jeel Patel's skills, projects, certifications, and education in the cybersecurity domain. It is designed with a premium off-white and orange color palette, smooth animations, and a recruiter-friendly layout. The site is live on Vercel and automatically updates on every GitHub push.

5. Key Features

  • Typing Animation — Hero title cycles through: Cybersecurity Learner | Network Security | Hardware Troubleshooting with blinking cursor
  • Light / Dark Mode — Theme toggle button in navbar using CSS variables
  • Animated Statistics — Counting animation for "5+ Projects" and "3+ Certifications"
  • Scroll Animations — Elements fade in as the user scrolls down the page
  • Mouse Glow Effect — Subtle glow that follows the mouse cursor
  • Fully Responsive — Optimized for mobile, tablet, and desktop screens
  • Download Resume — CTA button links directly to the resume PDF
  • View Projects — Button navigates to the dedicated Projects page
  • Floating Badge — Animated floating badge on the profile image
  • Cybersecurity Grid Background — Subtle orange grid pattern in the hero section
  • Profile Glow — Pulsing orange glow effect behind the profile image

8. How to Update the Portfolio

Whenever you want to update your live portfolio, follow these 3 steps:

  1. Make changes to the code in VS Code
  2. Test locally using npm run dev
  3. Push to GitHub using the VS Code terminal:
git add .
git commit -m "Describe your change here"
git push

Vercel will automatically detect the push and rebuild your live site within ~60 seconds.
